## 4. 验证证据要求

- UI 问题建议保留截图、浏览器控制台报错、网络请求状态和复现账号。
- 表单类问题需记录输入值、Zod 错误、接口响应、按钮状态，以及 URL 是否泄露表单字段。
- 移动端问题需记录 viewport 尺寸。
- 已知缺陷复现成功后，同步更新 `known-issues.md` 的实际结果和状态。
