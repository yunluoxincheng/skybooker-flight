# 已知缺陷清单

当前清单来自用户交付前测试准备阶段反馈。2026-07-07 已完成 KI-001 至 KI-009 的浏览器与接口复现抽样，并已为确认缺陷创建 GitHub Issue。未覆盖到的同类页面仍需后续扩展验证，不得在未复测前标记为已修复或通过。

## 严重级别

| 级别 | 标准 |
|---|---|
| S0 阻塞 | 系统无法启动、主流程完全不可用、无法登录、无法下单、严重数据损坏 |
| S1 严重 | 核心功能错误、安全越权、支付/订单状态错误、用户数据泄露 |
| S2 一般 | 非核心功能异常、部分筛选错误、提示不准确、普通页面交互问题 |
| S3 轻微 | 样式细节、文案、间距、图标、轻微体验问题 |
