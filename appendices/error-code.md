# 错误码规范

| 错误码 | 说明 |
|---|---|
| 200 | 成功 |
| 10001 | 用户未登录 |
| 10002 | 用户无权限 |
| 10003 | 邮箱格式不正确 |
| 10004 | 邮箱已注册 |
| 10005 | 验证码错误或已过期 |
| 10006 | 验证码发送过于频繁 |
| 10007 | 邮箱或密码错误 |
| 10008 | 账号已被禁用 |
| 10009 | 两次输入的密码不一致 |
| 10010 | 密码强度不足 |
| 10011 | Token 已失效 |
| 10012 | 账号类型不允许登录当前入口 |
| 20001 | 航班不存在 |
| 20002 | 座位已被占用 |
| 20003 | 座位不可选 |
| 20004 | 航班已取消 |
| 30001 | 订单不存在 |
| 30002 | 订单状态不允许当前操作 |
| 30003 | 订单已超时 |
| 30004 | 重复请求已处理 |
| 40001 | 候补订单不存在 |
| 40002 | 当前航班仍有余票，无需候补 |
| 40003 | 候补订单已过期 |
| 40004 | 候补订单状态不允许当前操作 |
| 50001 | AI 参数解析失败 |
| 50002 | AI 推荐结果为空 |
| 90000 | 系统异常 |
